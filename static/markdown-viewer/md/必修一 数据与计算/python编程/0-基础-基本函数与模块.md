## 数学模块与随机数模块
- `math`数学模块
- `random`随机数模块

**这两个高中要求的模块中，封装了大量函数方便我们使用**
### 使用方法
先通过`import`进行导入才可以使用
```python
import math
```
然后用`math.`的形式调用函数
>输出2
```python
import math
print(math.ceil(1.1)) #调用math模块的向上取整函数
```
## 四、数学函数（`math` 模块）

| 函数                      | 作用                       | 示例                   |
|--------------------------|----------------------------|------------------------|
| `math.sqrt(x)`           | 求平方根                  | `math.sqrt(16)`        |
| `math.pow(x, y)`         | 计算 `x` 的 `y` 次幂     | `math.pow(2, 3)`       |
| `math.floor(x)`          | 向下取整                  | `math.floor(3.9)`      |
| `math.ceil(x)`           | 向上取整                  | `math.ceil(3.1)`       |
| `math.pi`                | 圆周率常量 `π`            | `math.pi`              |
| `math.e`                 | 自然常数 `e`              | `math.e`               |

---

## 四、随机数函数（`random` 模块）

| 函数                      | 作用                                | 示例                        |
|--------------------------|-------------------------------------|-----------------------------|
| `random.randint(a, b)`  | 返回 `[a, b]` 之间的随机整数      | `random.randint(1, 10)`    |
| `random.random()`        | 返回 `[0.0, 1.0)` 之间的浮点数   | `random.random()`          |
| `random.choice(seq)`     | 从序列中随机选择一个元素          | `random.choice([1, 2, 3])` |
| `random.shuffle(seq)`    | 打乱序列中的元素顺序              | `random.shuffle(my_list)`  |

# **经典用法示例**

### 例 1：计算两个数的平方和
```python
import math

a = int(input("请输入第一个整数: "))
b = int(input("请输入第二个整数: "))
result = math.pow(a, 2) + math.pow(b, 2)
print("两个数的平方和为:", result)
```

---

### 例 2：判断一个整数是否为小数
```python
number = float(input("请输入一个数: "))

if number == int(number):
    print("该数是整数。")
else:
    print("该数是小数。")
```

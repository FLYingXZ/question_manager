# 自定义函数

在 Python 中，**函数**是一段可重复使用的代码，用于执行特定的任务。我们可以通过定义自定义函数来提高代码的组织性、可读性和复用性。自定义函数有一个特定的格式，定义后就可以在程序的其他部分调用。

---

## 1. 自定义函数的基本格式

```python
def 函数名(参数1, 参数2, ...):
    # 函数体
    # 执行操作
    return 返回值
```

- `def` 是定义函数的关键字。
- `函数名` 是你为函数取的名字，遵循标识符规则（字母、数字、下划线，且不能以数字开头）。
- `参数1, 参数2, ...` 是函数的输入，可以有多个，也可以没有。
- `return` 关键字用于返回函数的结果。如果没有 `return`，函数会返回 `None`。

---

## 2. 示例：没有参数和返回值的函数

这是最简单的一种函数，它不接收任何参数，也不返回任何结果。

```python
def greet():
    print("Hello, welcome to Python!")
```

调用该函数：

```python
greet()  # 输出: Hello, welcome to Python!
```

---

## 3. 示例：有参数但没有返回值的函数

这种函数会接收一些输入，但只进行一些操作，不返回结果。

```python
def greet_person(name):
    print("Hello, " + name + "!")
```

调用该函数：

```python
greet_person("Alice")  # 输出: Hello, Alice!
```

---

## 4. 示例：有参数且有返回值的函数

这种函数不仅接收参数，还返回结果。

```python
def add(a, b):
    return a + b
```

调用该函数并获取返回值：

```python
result = add(3, 5)  # result 等于 8
print(result)        # 输出: 8
```

---

## 5. 函数的返回值

- 如果没有 `return`，函数默认返回 `None`。
- 可以返回多种类型的数据，例如字符串、数字、列表等。

例如，返回一个计算结果：

```python
def multiply(x, y):
    return x * y
```

## 9. 小结

自定义函数可以使代码更加模块化和高效。你可以根据需要定义函数，传递参数，接收返回值，甚至设定默认参数和变长参数。掌握函数的定义和使用是 Python 编程的基础。

## 10.扩展阅读
### 自定义函数的常见应用

| 功能          | 示例代码                                    | 说明                               |
|---------------|---------------------------------------------|------------------------------------|
| 无参数无返回值 | `def greet():`<br> `    print("Hello!")`     | 用于简单的输出操作                 |
| 有参数无返回值 | `def greet_person(name):`<br> `    print(name)`| 用于传入参数进行处理               |
| 有参数有返回值 | `def add(a, b):`<br> `    return a + b`      | 用于返回计算结果                   |
| 默认参数值    | `def greet(name="Guest"):`<br> `    print(name)`| 如果调用时不传参数，则使用默认值 |
| 关键字参数    | `def greet(name, age):`<br> `    print(name, age)`| 通过参数名传递参数，增强可读性    |
| 变长参数      | `def print_info(*args, **kwargs):`<br> `    print(args, kwargs)`| 接收任意数量的位置或关键字参数   |
### 默认参数值

在定义函数时，可以为参数设置默认值，这样调用函数时可以不传递参数，使用默认值。

```python
def greet(name="Guest"):
    print("Hello, " + name + "!")
```

调用时，可以不传递参数：

```python
greet()  # 输出: Hello, Guest!
```

也可以传递参数：

```python
greet("Alice")  # 输出: Hello, Alice!
```

---

### 关键字参数

通过传递参数时，指定参数的名称来增强代码的可读性。

```python
def greet(name, age):
    print("Name: " + name + ", Age: " + str(age))
```

使用关键字参数传递：

```python
greet(age=18, name="Bob")  # 输出: Name: Bob, Age: 18
```

---

### 变长参数

有时候我们不确定一个函数需要接收多少个参数。Python 提供了两种方法来实现变长参数：

- `*args`：接收多个位置参数，并将其封装成一个元组。
- `**kwargs`：接收多个关键字参数，并将其封装成一个字典。

```python
def print_info(*args, **kwargs):
    print("Positional arguments:", args)
    print("Keyword arguments:", kwargs)
```

调用：

```python
print_info(1, 2, 3, name="Alice", age=25)
# 输出:
# Positional arguments: (1, 2, 3)
# Keyword arguments: {'name': 'Alice', 'age': 25}
```

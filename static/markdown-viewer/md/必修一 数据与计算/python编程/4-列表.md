# 4-列表

## 【功能】

存储多个数据，例如整数数列 `12, 23, 5, 18, 26, 13`。

## 【使用说明】

**1. 如何存储？**
 列表以 `[]` 将多个数据以逗号分隔的形式存储：

```python
a = [12, 23, 5, 18, 26, 13]
```

**2. 如何输入？**
 通过空格分隔，将一行整数输入到列表 `a`：

```
a = list(map(int, input().split(" ")))
```

**3. 如何输出？**
 与字符串类似：

- 第一个元素用 `a[0]` 表示；
- 切片操作可以用 `a[0:3]` 对列表进行切割。

## 扩展
列表的`懒人用法`（学考不要求，尽量不要出现在试卷答案中）
#### 末尾添加
```
a = [1, 2, 3, 4, 5]
#append(x)添加内容x
a.append(100)
print(a) #结果为[1, 2, 3, 4, 5, 100]
```
#### 任意位置插入
```
a = [1, 2, 3, 4, 5]
#insert(x, y)在位置x插入内容y
a.insert(1, 100)
print(a) #结果为[1, 100, 2, 3, 4, 5]
```
#### 任意位置删除
```
a = [1, 2, 3, 4, 5]
#pop(x)删除位置x的内容
a.pop(1)
print(a) #结果为[1, 2, 3, 4, 5]
```
#### 查询所在位置
```
a = [1, 2, 3, 4, 5]
#index(x)查询x的内容
print(a.index(4)) #结果为3
```
#### 最大最小值
```
a = [1, 2, 3, 4, 5]
#max(a)得到最大值
print(max(a)) #结果为5
```

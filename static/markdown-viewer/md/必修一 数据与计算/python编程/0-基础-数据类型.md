# 0-基础-数据类型

## 1. 什么是数据类型

在编程中，数据类型指的是变量中存储的数据的种类。每种数据类型有其特定的操作方式和存储需求。不同的数据类型可以让程序在执行时高效地进行计算和存储。

常见的基本数据类型包括：整数、浮点数、字符、布尔值等。

### 数据类型分类

| 分类           | 描述                                          | 示例                   |
|----------------|---------------------------------------------|------------------------|
| **整数**       | 用于表示没有小数部分的数字。                     | `3`, `-10`, `0`         |
| **浮点数**     | 用于表示带有小数部分的数字。                      | `3.14`, `-2.5`, `0.0`   |
| **字符**       | 用于表示单个字符。                               | `'A'`, `'b'`, `'9'`     |
| **布尔值**     | 用于表示真（True）或假（False）。               | `True`, `False`         |
| **字符串**     | 一系列字符组成的文本。                           | `"Hello"`, `"Python"`   |
| **列表**       | 有序的元素集合，可以包含不同类型的元素。           | `[1, 2, 3]`, `['a', 3]` |

## 2. 数据类型详解

### 2.1 整数 (int)
整数用于表示没有小数部分的数字，可以是正数、负数或零。

- 示例：`42`, `-100`, `0`
- 在 Python 中，整数没有大小限制，通常只有内存限制。

```python
x = 5
y = -12
z = 0
```

### 2.2 浮点数 (float)
浮点数表示带有小数部分的数字。

- 示例：`3.14`, `-0.001`, `100.0`
- 浮点数可以表示非常大或非常小的数，但在计算中有精度问题。

```python
a = 3.14
b = -2.5
c = 0.0
```
- 保留小数
```python
pi = 3.14159
print(round(pi))
#输出3
print(round(pi,3)) #第二个参数可设定保留位数
#输出3.142
```

### 2.3 字符串 (str)
字符用于表示字符，通常用单引号'或者双引号"括起来。

- 示例：`'A'`, `'b'`, `'3'`

```python
char_example = 'ABCD'
char_example = "ABCD"
```

### 2.4 布尔值 (bool)
布尔值表示逻辑上的真（True）或假（False）。

- 示例：`True`, `False`
- 布尔值在条件判断语句中非常重要。

```python
flag = True
```

### 2.5 列表 (list)
列表是一个有序的元素集合，可以包含不同类型的元素。

- 示例：`[1, 2, 3]`, `['a', 'b', 3]`
- 列表中的元素是可变的，可以改变、添加或删除元素。

```python
my_list = [1, 2, 3, 4]
my_list.append(5)
print(my_list)
#输出[1,2,3,4,5]
print(my_list[0])
#输出1
```

## 3. 数据类型转换

在 Python 中，不同类型的数据可以相互转换。例如：

- 整数转浮点数：`float(3)`
- 浮点数转整数：`int(3.5)`
- 字符串转整数：`int("123")`
- 整数转字符串：`str(123)`

```python
x = 3
y = float(x)  # 转换为浮点数并赋值到y
z = str(x)    # 转换为字符串并赋值到z
```

## 4. 小结

理解数据类型是编程中的基础，不同的数据类型有不同的操作方法，掌握它们能够有效提高编程效率。掌握数据类型的同时，了解如何进行数据类型转换也是非常重要的。

## 扩展阅读

 尴尬时刻：当数据类型“不按套路出牌”

### 1. 浮点数的“数学课代表也会算错”
**现象**：你让计算机算 `0.1 + 0.2`，它告诉你等于 `0.30000000000000004`
```python
print(0.1 + 0.2)  # 输出：0.30000000000000004
```

**原理**（说人话版）：
- 计算机是二进制思维，而我们习惯十进制
- 就像1/3在十进制里是0.33333...永远除不尽一样，有些十进制小数在二进制里也“除不尽”
- 计算机只能存个近似值，所以两个近似值相加，误差就露馅了

**解决手段**：
```python
# 方法1：四舍五入，眼不见为净
print(round(0.1 + 0.2, 2))  # 输出：0.3

# 方法2：用Decimal（专门处理钱的库）
from decimal import Decimal
print(Decimal('0.1') + Decimal('0.2'))  # 输出：0.3

# 方法3：比较时用容差，别问是不是完全相等
abs(0.1 + 0.2 - 0.3) < 0.000001  # True
```

### 2. 字符串和数字的“身份困惑”
**现象**：你以为它是数字，其实它是戏精
```python
age = input("请输入年龄：")  # 用户输入：25
# 下面这行会报错，因为age是字符串"25"，不是数字25
print("明年你就" + (age + 1) + "岁了")  # 报错！
```

**原理**：
- `input()`函数给你的都是字符串，就像给你一盒写着“25”的饼干，你不能吃纸盒
- 字符串的`+`是拼接，不是加法

**解决手段**：
```python
# 方法1：明确转换身份
age = int(input("请输入年龄："))  # 把字符串"25"变成数字25

# 方法2：温柔一点，先试试能不能转
user_input = input("请输入数字：")
if user_input.isdigit():  # 检查是不是纯数字
    num = int(user_input)
else:
    print("你在逗我吗？")
```

### 3. 列表的“共享经济”翻车
**现象**：改了一个，全变了
```python
a = [1, 2, 3]
b = a  # 你以为这是复制，其实是“共享账号”
b.append(4)
print(a)  # 输出：[1, 2, 3, 4] 咦？我没改a啊！
```

**原理**：
- `b = a` 只是让b指向a的同一个“内存地址”（就像给你的微信起了个外号）
- 改b其实就是改a，因为它们共享同一个家

**解决手段**：
```python
# 方法1：真正复制一份
a = [1, 2, 3]
b = a.copy()  # 或者 b = a[:]（切片复制）
b.append(4)
print(a)  # [1, 2, 3] ✓
print(b)  # [1, 2, 3, 4]

# 方法2：对于多层嵌套列表，用深拷贝
import copy
a = [[1, 2], [3, 4]]
b = copy.deepcopy(a)  # 连孙子辈都复制
```

### 4. 布尔值的“万物皆可判断”
**现象**：Python觉得很多东西都是False
```python
# 这些在if判断里都相当于False：
if 0:        # 数字0
if "":       # 空字符串
if []:       # 空列表
if None:     # 空值
if False:    # 布尔False本身

# 这些相当于True：
if -1:       # 非0数字
if "False":  # 非空字符串（包括"False"这四个字母）
if [0]:      # 非空列表
```

**原理**：Python的设计哲学是“明确比隐晦好”，但为了写代码方便，允许这种隐式转换

**避坑指南**：
```python
# 比较数字时，明确写出来
if x == 0:      # ✓ 清晰
if not x:       # ✗ 模糊，x可能是0、None、空列表...

# 检查列表是否为空
if len(my_list) == 0:   # ✓ 清晰但啰嗦
if not my_list:         # ✓ Pythonic写法
if my_list:             # ✓ 非空时执行，更优雅
```

## 数据类型生存法则

1. **“别猜，要问”**：不确定类型时，用`type()`函数看看
   ```python
   print(type(3.14))    # <class 'float'>
   print(type("hello")) # <class 'str'>
   ```

2. **“先试后转”**：转换类型前先判断能不能转
   ```python
   try:
       num = int(user_input)
   except ValueError:
       print("这不是个有效的数字！")
   ```

3. **“亲兄弟明算账”**：需要独立副本时，记得用`.copy()`或切片

4. **“浮点数别比相等”**：比较浮点数用容差范围，别用`==`

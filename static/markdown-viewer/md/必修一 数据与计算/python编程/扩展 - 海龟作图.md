# Python3海龟作图学习文档

## 简介

turtle是Python内置的绘图库，它利用计算机编程来控制小海龟在屏幕上画图，实现了一个简单的绘图工具。

## 初识turtle

###  1.生成小海龟

在使用turtle之前，需要先导入turtle库。

```python
import turtle
```

第一步，生成一只海龟

```python
t = turtle.Turtle()
```

好了，以后这个 t 就是我们的小海龟，运行试试？

![image-20260114210512765](https://flying.zj.cn/_uploads/photos/EMywwNnmRj12MD9H.png)

什么？你说不是小海龟，试试加一句这个

```python
t.shape("turtle") #改变小海龟的形状
```

还能改颜色哦

```python
t.color("blue")  #改变小海龟的颜色
```

### 2.海龟动起来

为了控制海龟的运动，turtle库提供了很多函数，如`forward、backward、left、right`等方法可以控制小海龟向前、向后、向左、向右移动。

| 函数名            | 功能          |
| :---------------- | :------------ |
| `t.forward(100)`  | 前进 100 像素 |
| `t.backward(100)` | 后退 100 像素 |
| `t.left(90)`      | 向左转90°     |
| `t.right(90)`     | 向右转90°     |

例如以下代码将小海龟向前移动100像素，再向右转90度，再向前移动100像素。

```python
t = turtle.Turtle()
t.forward(100) #向前100个单位
t.right(90)    #右转90°
t.forward(100) #向前100个单位
```

![image-20260114210534568](https://flying.zj.cn/_uploads/photos/YSnSip2qSuIKh1lA.png)

什么？嫌弃小海龟跑得太慢了？

试试

```python
t.speed(0)
```

这将是一个让兔子见了都摇头的速度

什么？还不够快？用`goto()`

`goto`方法，可以将海龟移动到指定的坐标位置，并在当前位置与目标位置之间绘制直线。

例如以下代码绘制一条从当前位置到(100,100)的线段。

```python
t.goto(100,100)
```

![image-20260114210602687](https://flying.zj.cn/_uploads/photos/wEbGcm13V837ZGWd.png)

当然，有时候我们也希望小海龟可以不产生移动痕迹，那么我们可以用（提笔/下笔）两个函数

```python
t.penup()  #提笔，如此则小海龟的移动将不留下痕迹
t.pendown()#下笔，小海龟的移动留下痕迹
#注意默认是下笔状态
```

### 3.绘制圆

`circle`方法用于绘制圆形，该方法的参数为半径及角度，例如以下代码绘制一个半径为50的圆。

```python
t.circle(50) #半径为50的圆
```

```python
t = turtle.Turtle()
t.circle(50,180) #半径为50，弧度为180°的圆（即半圆）
```

旋转角度是常用的函数，用好了可以产生许多美妙的图片效果

例如：

```python
import turtle

t = turtle.Turtle()
t.speed(0)
for i in range(36):
    t.circle(100)
    t.right(10)
```

### 4.绘制多边形

turtle库提供了方法来绘制多边形，方法的参数为边的长度及边数，例如以下代码绘制一个边长为100的正五边形。

```python
t.forward(100)
t.right(72)
t.forward(100)
t.right(72)
t.forward(100)
t.right(72)
t.forward(100)
t.right(72)
t.forward(100)
```

![image-20260114210634174](https://flying.zj.cn/_uploads/photos/17LMUiRZYE780eZA.png)

显然这种方法有一点麻烦，试试用```for```循环改写？

```python
for i in range(5):
    t.forward(100)
    t.right(72)
```

说到底，只要有编程的地方就有循环，循环才是精髓啊。

### 5.颜色填充

黑白色太单调了，没关系，彩色当然可以支持。

利用`t.begin_fill()`和`t.end_fill()`，两条语句之间所绘制的封闭图形将被填充成指定的颜色

```python
t = turtle.Turtle()
t.fillcolor("red") #设置填充颜色
t.begin_fill()     #开始填充
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)
t.end_fill()       #结束填充
```

画笔的颜色也是能改的

```python
t.pencolor('red')
```



## 样例图形

通过组合基本图形，可以实现更复杂的图形设计。下面介绍几个例子。

### 绘制弹簧

以下代码绘制一朵花，其中包含五个花瓣，每个花瓣由四个半圆构成。

```python
t = turtle.Turtle()
for i in range(5):
    t.circle(50,180)
    t.circle(20,180)
t.hideturtle() #画好后隐藏小海龟
```

### 绘制五角星

以下代码绘制一个蓝色的五角星。

```python
t = turtle.Turtle()
t.color("blue")
for i in range(5):
    t.forward(100)
    t.right(144)
t.hideturtle() #画好后隐藏小海龟
```

### 绘制渐变色环

以下代码绘制一个渐变色环。

```python
t = turtle.Turtle()
colors = ["red","orange","yellow","green","blue","indigo","purple"]
for i in range(50):
    t.color(colors[i%7])
    t.circle(100)
    t.right(7.2)
t.hideturtle()
```

## 可能用到的操作

| 函数名           | 功能                             | 例子                                     |
| :--------------- | :------------------------------- | :--------------------------------------- |
| `t.forward()`    | 向当前方向移动指定距离的像素     | `t.forward(100)`                         |
| `t.backward()`   | 向当前相反方向移动指定距离的像素 | `t.backward(50)`                         |
| `t.left()`       | （向左转）当前方向一定的度数     | `t.left(90)`                             |
| `t.right()`      | （向右转）当前方向一定的度数     | `t.right(45)`                            |
| `t.penup()`      | 抬起画笔，不产生画线             | `t.penup()`                              |
| `t.pendown()`    | 落下画笔，开始画线               | `t.pendown()`                            |
| `t.setpos()`     | 移动到指定位置                   | `t.setpos(-100,100)`                     |
| `t.pensize()`    | 设置画笔的宽度                   | `t.pensize(5)`                           |
| `t.pencolor()`   | 设置画笔的颜色                   | `t.pencolor('red')`                      |
| `t.fillcolor()`  | 设置填充的颜色                   | `t.fillcolor('yellow')`                  |
| `t.begin_fill()` | 开始进行图形填充操作             | `t.begin_fill()`                         |
| `t.end_fill()`   | 结束图形填充操作                 | `t.end_fill()`                           |
| `t.circle()`     | 以指定半径绘制圆形               | `t.circle(50)`                           |
| `t.dot()`        | 绘制指定大小和颜色的圆点         | `t.dot(10, 'blue')`                      |
| `t.write()`      | 将文本写入画布中的指定位置       | `t.write('Hello World', align='center')` |

希望这个表格能够帮助你更好地了解海龟作图的基本操作和用法。

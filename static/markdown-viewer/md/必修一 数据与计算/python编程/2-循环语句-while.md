# 2-循环语句-while

`while` 循环是 Python 中的一个控制结构，它允许我们反复执行一段代码，直到某个条件不满足为止。

## 知识框架

- **定义**：`while` 循环根据条件判断是否继续执行循环体。
- **语法**：
  ```python
  while 条件:
      循环体
  ```
  - `条件`：一个布尔表达式，返回 `True` 或 `False`。
  - `循环体`：一段会在条件为 `True` 时重复执行的代码。

## 主要特点
1. **条件判断**：循环在每次执行前都会判断条件表达式。如果条件为 `True`，执行循环体；如果为 `False`，退出循环。
2. **循环控制**：
   - 使用 `break` 语句可以立即跳出循环。
   - 使用 `continue` 语句跳过当前循环的剩余部分，直接进行下一轮条件判断。
   
## 示例代码

### 1.基本示例

```python
# 从1打印到5
i = 1
while i <= 5:
    print(i)
    i += 1  # 递增i，避免死循环
```

**解释**：
- 初始时 `i` 为 1。
- 每次执行循环体后，`i` 会加 1。
- 当 `i` 大于 5 时，循环条件 `i <= 5` 为 `False`，退出循环。

### 2.使用 `break` 跳出循环

```python
# 当输入数字为 0 时，跳出循环
while True:
    num = int(input("请输入一个数字（0退出）："))
    if num == 0:
        break
    print("你输入的数字是：", num)
```

**解释**：
- `while True` 创建了一个无限循环。
- 输入为 0 时，使用 `break` 跳出循环。

### 3.使用 `continue` 跳过当前循环

```python
# 打印 1 到 5，跳过 3
i = 1
while i <= 5:
    if i == 3:
        i += 1
        continue
    print(i)
    i += 1
```

**解释**：
- 当 `i == 3` 时，`continue` 跳过当前循环的剩余部分，继续执行下一轮循环。

### 4.输入任意个数的数字，直到输入0时结束，输出总和
```python
#写法一
Sum = 0
while True: #先令其无限循环
	n = int(input()) #输入写在循环中，才能做到多次输入
	Sum += n
	if n==0:
		break
#写法二
n = int(input()) #先输入第一个数字
Sum = n
while n!=0: #当n不为0时反复输入
	n = int(input())
	Sum += n
print(Sum)
```
请思考，两种写法的本质其实是一样的

### 5.猜数游戏
编写程序，由系统随机生成1~100的数字，玩家可以反复猜，系统每次给出“偏大/偏小”的提示，直到猜中。
```python
import random
ans = random.randint(1, 1000) #用于生成随机数，每次不一样
n = 0 #定义玩家输入的数字变量
while n!=ans:  #当没猜中的时候进入循环
	n = int(input())
	if n > ans:
		print("偏大")
	elif n < ans:
		print("偏小")
	else:
		print("猜中了！！！")
```

  | 特性         | while 循环   | for 循环               |
  | ------------ | ------------ | ---------------------- |
  | **循环条件** | 基于条件 | 基于范围         |
  | **循环次数** | 通常不确定   | 通常确定               |
  | **适用场景** | 条件驱动循环 | 遍历序列、固定次数循环 |
许多情境下while与for循环可以相互转化。例如：
> 从1循环到10

```python
#for循环:
for i in range(1, 11):
	print(i)
#while循环:
i = 1
while i < 11:
	print(i)
	i += 1
```
> 若数字x为偶数，则//2，否则是奇数则+1，直到x为1停止循环

```python
#for循环（不推荐）：
x = int(input())
for i in range(1000000000): #由于不知道具体循环次数，只能设置一个尽可能大的循环范围
	if x%2==0:
		x //= 2
	else:
		x += 1
	if x==1:
		break
#while循环:
x = int(input())
while x != 1:
	if x%2==0:
		x //= 2
	else:
		x += 1
```


## 常见错误

1. **死循环**：如果 `while` 条件一直为 `True`，循环将永远不会停止，导致程序卡住。比如：
   ```python
   while True:
       print("这将是一个死循环")
   ```
   为避免死循环，必须确保条件能够变为 `False` 或者在某些情况下使用 `break` 跳出。

2. **条件未更新**：如果循环体内部没有更新条件，可能导致意外的行为。例如：
   ```python
   i = 1
   while i <= 5:
       print(i)
       # 这里忘记更新 i，导致死循环
   ```

## 扩展阅读

- `while` 循环和 `for` 循环：`while` 循环适合于条件判断，而 `for` 循环适用于已知迭代次数或遍历可迭代对象的场景。
- **嵌套循环**：`while` 循环可以嵌套在其他循环或条件语句内。
  
## 小结

- `while` 循环通过反复判断条件来执行循环体，适用于那些不确定执行次数的场合。
- 在使用 `while` 循环时要特别注意条件的更新，避免进入死循环。

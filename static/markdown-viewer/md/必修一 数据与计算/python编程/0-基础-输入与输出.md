# 0-基础-赋值语句

## 1. **什么是输入与输出？**

在编程中，输入（input）和输出（print）是最基本的概念之一。程序需要与用户进行交互，接收用户的输入信息，并将结果反馈给用户。

- **输入**：指程序从外部获取数据或信息。
- **输出**：指程序将处理后的结果显示给用户。

例如：

- 用户输入一个数字，程序根据这个数字进行计算，然后输出计算结果。

在 Python 中，输入和输出主要通过 `input()` 和 `print()` 函数来实现。

------

## 2. **Python 中的输入**


Python 提供了 `input()` 函数用于接收用户的输入。`input()` 函数会暂停程序执行，等待用户输入，然后返回输入的内容。

- **基本语法**：

  ```
  输入变量 = input()
```

- **说明**：

  - `input()` 函数的返回值是字符串类型，即使用户输入的是数字，返回的也是数字形式的字符串。

### 示例1——字符串的连接：

```
name = input()
print("你好，" + name + "!")
```

**运行结果**（用户输入"fly"）：

```
fly
你好，fly!
```

### 示例2——带有提示的输入：
```python
name = input("请输入姓名：")
print("你好，" + name + "!")
```

**运行结果**：

```
请输入姓名：fly
你好，fly!
```

**说明：**`input("请输入姓名：")`可以在输入时显示括号中的内容作为提示，类似于：
```python
print("请输入姓名：")
name = input()
```
请注意，在平台提交时请勿使用该方法，否则结果一定会显示错误！！！
### 示例3——输入多项内容：

- **输入两个姓名：**

  ①分两行输入

  ```
  name1 = input()
  name2 = input()
  print(name1,name2)
  #输入:
  fly
  mike
  #输出：
  fly mike
```

  ②以空格为间隔一行输入

  ```
  name1, name2 = input().split(" ")
  #输入：
  fly mike
  #输出：
  fly mike
```
  ③以空格为间隔一行输入多个整数
  ```
  a,b=map(int,input().split())
  c=a+b
  print(c)
```

### 2.2 **转换输入类型**

虽然 `input()` 返回的是字符串，但我们可以通过类型转换，将输入的字符串转化为其他数据类型，例如整数（`int`）或浮点数（`float`）。

对于`数据类型`，可对应的章节学习。

#### 转换为整数：

  ```
  num = int(input())
```

#### 转换为浮点数（小数）：

  ```
  num = float(input())
```

#### 示例：

```
# 输入两个数字并进行加法运算
num1 = int(input())
num2 = int(input())
sum = num1 + num2
print("两个数字的和是："+str(sum))
```

**运行结果**：

```
5
7
两个数字的和是： 12
```

## 3. **输出**

Python 使用 `print()` 函数来输出信息。`print()` 函数会将括号内的内容显示在屏幕上。

- **基本语法**：

  ```python
  print(内容)
```

- **说明**：

  - `print()` 函数可以输出字符串、数字、变量的值等。
  - 默认情况下，`print()` 会在输出的内容后添加一个换行符。如果不希望添加换行符，可以通过 `end` 参数来控制。

### 3.1 **简单输出**

#### 示例：

```
print("Hello, World!")  # 输出一个简单的字符串
print(100)               # 输出一个整数
```

**运行结果**：

```
Hello, World!
100
```

### 3.2 **输出多个内容**

`print()` 可以同时输出多个内容，它们之间会默认用空格分隔。

#### 示例：

```python
name = "小明"
age = 18
print("姓名:",name,"年龄:",age)
```

**运行结果**：

```
姓名: 小明 年龄: 18
```
细心的你会注意到，以上的输出会使得内容之间有一个空格，这是Python默认的机制
如何做到没有空格呢？可以用之前学习过的类型转换函数，将所有数字变量转变为字符串后，用+连接
```
name = "小明"
age = 18
print("姓名:"+str(name)+"年龄:"+str(age)
```
### 3.3 **不换行输出**

默认情况下，`print()` 会输出内容后换行。如果我们希望输出内容不换行，可以使用 `end` 参数。

#### 示例：

```python
print("Hello", end=",")
print("World!")
```

**运行结果**：

```
Hello,World!
```

在这个例子中，`end=","` 使得第一个 `print()` 输出后的换行符被逗号替代。

## 4. **输入与输出的结合使用**

在实际编程中，我们经常会将输入和输出结合起来使用。通过输入获取用户数据，然后用 `print()` 输出处理结果。

### 示例：计算圆的面积

```python
# 输入半径
radius = float(input("请输入圆的半径："))
# 计算面积
area = 3.14 * radius * radius
# 输出结果
print("圆的面积是：", area)
```

**运行结果**：

```
请输入圆的半径：5
圆的面积是： 78.5
```

------

## 5. **总结**

- **输入（input）**：从用户那里获取数据，返回值为字符串类型。
- **输出（print）**：将信息显示给用户，支持多个内容的输出，支持格式化输出。

这两个概念是编程中最基础的部分，它们帮助我们与用户进行互动，进行信息传递。

## 6. **扩展阅读：**

### **格式化输出**

Python 提供了不同方式进行格式化输出：

- **使用 `+` 连接字符串**：

  ```
  name = "小明"
  age = 18
  print("姓名: " + name + "，年龄: " + str(age))  # 注意：age 需要转换为字符串
```

- **使用 f-string（常用方式，但是高中教材不要求）**：

  ```
  name = "小明"
  age = 18
  print(f"姓名: {name}，年龄: {age}")
```

- **使用 `format()` 方法（常用方式，但是高中教材不要求）**：

  ```
  name = "小明"
  age = 18
  print("姓名: {}，年龄: {}".format(name, age))
```

  **利用 `format()` 你还可以保留小数位数：**

  ```python
  number = 3.14159
  result = "{:.2f}".format(number)  # 保留两位小数
  print(result)  # 输出 3.14
  ```

### **文件输入输出**

有时候，需要将文件中的内容作为输入，而不是从键盘上输入，python也当然支持从文件读取内容

#### **文件的基本操作步骤**

1. **打开文件**：使用 `open()` 函数，指定文件名和模式。
2. **读取或写入文件**：使用相应的方法进行文件操作。
3. **关闭文件**：使用 `close()` 方法关闭文件，释放资源。

#### **打开文件模式**

| 模式 | 说明                                                                 |
|------|----------------------------------------------------------------------|
| `r`  | 只读模式（默认），如果文件不存在会报错。                               |
| `w`  | 写入模式，如果文件不存在则创建，如果存在则覆盖原内容。                   |
| `a`  | 追加模式，如果文件不存在则创建，如果存在则在文件末尾追加内容。           |
| `x`  | 创建模式，如果文件存在则报错，用于创建新文件。                         |
| `b`  | 二进制模式，例如 `rb` 或 `wb`，用于处理非文本文件（如图片、音频等）。    |
| `t`  | 文本模式（默认），用于处理文本文件。                                   |
| `+`  | 读写模式，例如 `r+` 或 `w+`，可以同时进行读写操作。                    |

#### **读取文件**
以文件 example.txt 为例
- **读取整个文件**：
  ```python
  with open("example.txt", "r", encoding="utf-8") as f:
      content = f.read()
      print(content)
  ```
  **说明**：使用 `with` 语句可以自动关闭文件，无需手动调用 `close()`。

- **逐行读取**：
  ```python
  with open("example.txt", "r", encoding="utf-8") as f:
      for line in f:
          print(line.strip())  # strip() 去掉行末的换行符
  ```

- **读取所有行并存入列表**：
  ```python
  with open("example.txt", "r", encoding="utf-8") as f:
      lines = f.readlines()
      for line in lines:
          print(line.strip())
  ```

#### **写入文件**

- **写入文本**：
  ```python
  with open("output.txt", "w", encoding="utf-8") as f:
      f.write("Hello, World!\n")
      f.write("这是写入的第二行。")
  ```

- **追加文本**：
  ```python
  with open("output.txt", "a", encoding="utf-8") as f:
      f.write("\n这是追加的内容。")
  ```

#### **文件操作示例**

```python
# 写入文件
with open("data.txt", "w", encoding="utf-8") as f:
    f.write("Python 文件操作示例\n")
    f.write("第二行内容。")

# 读取文件
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print("文件内容：")
    print(content)
```

#### **注意事项**

1. **字符编码**：在处理中文或其他非 ASCII 字符时，建议使用 `encoding="utf-8"` 参数，避免乱码问题。
2. **文件路径**：如果文件不在当前目录，需要提供完整的文件路径。
